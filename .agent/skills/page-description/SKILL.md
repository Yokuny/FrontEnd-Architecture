Expert skill for creating professional, comprehensive content for maritime fleet management and operations software platforms. This skill provides domain expertise for SaaS systems serving the shipping, offshore, and maritime industries.

## Content Creation Guidelines

### Writing Style for Maritime SaaS
- **Professional & Technical**: Use industry-standard terminology (IMO, SOLAS, MARPOL, ISM, STCW)
- **Action-Oriented**: Focus on what users can do, results they achieve
- **Data-Driven**: Emphasize metrics, efficiency gains, cost savings, compliance
- **Multilingual Ready**: Structure content for easy translation (avoid idioms)

### Page Description Structure
1. **Primary Headline**: Clear, benefit-focused (what this feature does)
2. **Problem Statement**: Brief context on the challenge being solved
3. **Feature Overview**: 2-3 sentences explaining the capability
4. **Key Benefits List**: 3-5 bullet points (efficiency, compliance, cost, safety, visibility)
5. **Detailed Features**: Sub-sections with specifics
6. **Use Cases/Scenarios**: Real-world examples
7. **Technical Details** (if applicable): Integrations, data sources, compliance standards
8. **Call-to-Action**: Demo request, documentation link, trial signup

### Maritime Industry Terminology

**Vessel Operations**:
- DWT (Deadweight Tonnage), GRT/NRT (Gross/Net Registered Tonnage)
- LOA/LBP (Length Overall/Between Perpendiculars)
- Draft, Freeboard, Displacement
- Ballast/Laden condition
- Sea trial, Performance warranty

**Chartering**:
- Charterer, Shipowner, Disponent Owner
- Voyage Charter, Time Charter, Bareboat/Demise Charter
- Freight rate ($/ton, Worldscale for tankers)
- Laytime, Demurrage, Despatch
- Off-hire, On-hire surveys
- Charter Party (CP), Fixture, Recap

**Compliance & Regulations**:
- IMO (International Maritime Organization)
- SOLAS (Safety of Life at Sea)
- MARPOL (Marine Pollution)
- ISM Code (International Safety Management)
- STCW (Standards of Training, Certification and Watchkeeping)
- Flag State, Port State Control (PSC)
- Class Society (DNV, ABS, LR, BV, ClassNK)

**Performance Metrics**:
- EEOI (Energy Efficiency Operational Indicator) = CO₂ / (cargo × distance)
- CII (Carbon Intensity Indicator) = CO₂ / capacity × distance
- EEXI (Energy Efficiency Existing Ship Index)
- Fuel consumption: MT/day (metric tons per day)
- Speed: Knots, RPM
- Specific Fuel Oil Consumption (SFOC): g/kWh

**Offshore/Oil & Gas Specific**:
- DP (Dynamic Positioning) operations
- PSV (Platform Supply Vessel), AHTS (Anchor Handling Tug Supply)
- ROV (Remotely Operated Vehicle) support
- Stand-by rates, Day rates
- Mobilization/Demobilization
- Utilization rate, Transit time
